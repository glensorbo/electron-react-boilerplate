# Electron React Boilerplate

## Comes with

- Electron
- Axios
- Moment
- React
- React Router
- React Moment
- React Select
- Redux Toolkit
- Tailwindcss
- Tailwind-scrollbar
- "Link" between React and Electron
- Update Electron App

## How to

```bash
npm install
```

```bash
npx tailwindcss -i ./src/index.css -o ./src/react/index.css --watch
```

```bash
npm start
```
